# C++ Data Structures and Algorithms Journey

This repository contains the source code, notes, and problem solutions developed during a 6-month intensive study of Data Structures, Algorithms, and Computer Science fundamentals using C++.

## Project Structure

The repository is organized by weekly topics. Each folder corresponds to a specific area of study and contains relevant C++ source files.

## Development Environment

- **Language**: C++ (Standard C++17 or later)
- **Compiler**: G++ (MinGW-w64)
- **Editor**: Visual Studio Code
- **Build System**: Manual compilation via terminal

## Compilation

To compile any source file in this repository, use the following command:

```bash
g++ filename.cpp -o output_name
./output_name
```
